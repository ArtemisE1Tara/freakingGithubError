Privacy policy

General

This document will tell you which data may get collected when you visit and use this site.

Logging by GitHub pages

As this page is hosted on GitHub pages GitHub will save some data about your visit for security resons. This may include:

- IP address
- The browser you're using
- The site you're coming From
- The time you visited this site
- Visited pages
- The operating system you use

To get an exact understanding of what is being logged reefer to GitHubs privacy policy

Logging by ComputerAnalytics

ComputerAnalytics is an analytics tools which provides information such as:

- Amount of clicks on the website
- Where visitors came from
- How long they were on the page and the time they visited it
- The page you visited
- Your devices screen height and width (which may or may not be accurate)

ComputerAnalytics itself is open source and it's code can be viewed by everyone here
How ComputerAnalytics works:

- It records the things mentioned above.
- After you leave the page the data gets sent to ComputerAnalytics main server at https://analytics.rui2015.me/
- The data will not be associated with you

ComputerAnalytics privacy policy can be found here
A writeup containing information about how ComputerAnalytics works can be found here
The data is not being merged with any other data of you

Logging by Oculus

If you use pages such as AppVersions Oculus API is being used to gather most of the data.
This data will log the same basic things mentioned in the Logging by GitHub pages section at the top.
Oculus privacy policy can be found here.
Due to programming reasons the requests gets proxied (more or less like telling someone to say something to someone else) through Heroku which again logs the same things outlined in Logging by GitHub pages.

Logging by YouTube

Some pages may use YouTube to play videos. The privacy poliy of YouTube can be found at https://www.google.de/intl/de/policies/privacy

Cookies

ComputerAnalytics uses cookies to give you an unique identifier which anonymises you in its database
