For QuestAppVersionSwitcher
A guide specific for Downgrading can be found here
What is this tool?
This tool allows you to backup, restore and downgrade quest apps directly on your quest
Info: Is this tool piracy?
No. You're downloading from Oculus servers and those check that you own the game
Why do I have to login to downgrade
QuestAppVersionSwitcher needs to authenticate the download with oculus so it has you login. Your login information is being stored encrypted only on your quest.
Issues getting the app to open?
Restart your Quest and try to open it again.
Want a video tutorial?
There is currently no up to date tutorial on QuestAppVersionSwitcher
Download and install the app
Install SideQuest and search for QuestAppVersionSwitcher. Install the app, open it and allow permissions.
Downgrade games
Open the Downgrade tab. If it tells you to log in go to the Tools & Options tab, press the login button and follow all provided instructions.
Restore backups
Go to the Backup tab, change the app to the one you want to restore and select the backup you want to restore.
Hit the restore button and follow the provided steps.
Create backups
Ggo to the Backup tabGo to the Backup tab, change the app to the one you want to backup and put in a backup name. Simply hit Create backup and the backup will get created.
